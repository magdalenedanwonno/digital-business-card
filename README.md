 Digital Business Card 📇

A sleek, responsive, and mobile-first digital business card built with **React**. This project was developed as part of a deep dive into React component architecture, focusing on modularity, accessibility, and modern CSS layout techniques.



 🚀 Features

 **Component-Based Architecture**: Built using modular React components (`Info`, `About`, `Interests`, and `Footer`).
 **Mobile-First Design**: Optimized for all screen sizes using CSS Flexbox and responsive units (`max-width`, `90%`).
 **Accessibility**: Implements descriptive `alt` text and `aria-label` tags for screen reader compatibility.
 **Single Source of Truth**: Uses a centralized `data.js` file to manage all personal information, making updates effortless.
 **Modern Styling**: Features a dark-theme aesthetic with high-fidelity professional branding.

 🛠️ Tech Stack

 **React**: UI Library
 **JavaScript**: Logic and data handling
 **CSS3**: Custom styling with a focus on responsiveness
 **Font Awesome**: Iconography for social media links
 **Google Fonts**: "Inter" font for a clean, professional look

📁 Project Structure

src/

│   ├── Footer.jsx
│   ├── Info.jsx
│   └── Interests.jsx
├── App.js            # Main container component
├── App.css           # Global styles and layout
├── data.js           # Centralized user data
└── index.js          # Entry point for React


⚙️ How to Run

1. Clone the repo.

2. Run npm install to install dependencies.

3. Run npm start to view the project locally.

📷 Preview

The card features a professional headshot, interactive buttons for Email and LinkedIn, and a social footer with links to Twitter, LinkedIn, Instagram, and GitHub.
